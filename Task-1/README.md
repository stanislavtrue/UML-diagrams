# Система бронювання квитків
## Опис предметної обласі
Система бронювання квитків надає можливість користувачам переглядати доступні рейси, вибирати місця та оформляти бронювання. Користувач може створювати та переглядати власні бронювання, а адміністратор має можливість додавати, редагувати та видаляти рейси. Після створення бронювання система генерує квиток і реєструє оплату.
