## Опис станів
+ **Created** - бронювання створене, але ще не оплачене.
+ **Pending Payment** - очікується оплата. Перехід відбувається коли клієнт створив бронювання, але ще не оплатив.
+ **Paid** - бронювання оплачено. Перехід відбувається після того як оплата підтверджена.
+ **Confirmed** - бронювання підтверджено. Перехід відбувається після того як буде надіслано електронний квиток.
+ **Cancelled** - бронювання скасовано. Перехід відбувається якщо бронювання було скасовано системою, або користувачем.
